Eventos JS

Drop

Un elemento es soltado en un destino válido.

Drag

Un elemento o texto seleccionato está siendo arrastrado (cada 35ms).

Playing

La reproducción está lista para comenzar después de haber sido pausada o retrasada por falta de datos.

Touchmove

Un punto táctil se desplaza a lo largo de la superficie táctil.

Wheel

Un botón rueda de un dispositivo señalador se gira en cualquier dirección.

Mouseover

Se activa cuando el puntero del ratón pasa sobre un elemento.

Keydown

Se presiona una tecla en el teclado.

Change

El valor de un elemento input, select o textarea cambia.

Focus

Se activa cuando un elemento (como un input) gana el foco.

Resize

Se cambia el tamaño de la ventana del navegador.

Load

Un documento o recurso externo ha sido cargado completamente.

Dragover

Un elemento o texto seleccionado es arrastrado encima de un destino válido (cada 50ms).

Dragend

Una operación de arrastre ha finalizado (al pulsar un botón del ratón o pulsando la tecla escape).

Canplaythrough

La aplicación de usuario puede reproducir los medios, y se estima que hay datos suficientes, han sido cargados para reproducir el medio hasta el final sin tener que parar para una mayor amortiguación del contenido

Unload

Se activa cuando un usuario abandona una página web.

Contextmenu

Se activa cuando se abre el menú contextual (clic derecho).

Scroll

Se activa cuando el usuario desplaza la barra de desplazamiento.

Keyup

Se activa cuando se suelta una tecla en el teclado.

Mouseup

Se activa cuando se suelta un botón del ratón.

Mouseout

Se activa cuando el puntero del ratón abandona un elemento.
